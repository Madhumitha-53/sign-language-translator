About the Project

The Sign Language Translator is an innovative application that uses computer vision and machine learning to translate sign language gestures into text in real-time. This project aims to bridge communication gaps and make sign language more accessible to everyone.

Technologies Used

  • Computer Vision (OpenCV)
  • Machine Learning (MediaPipe)
  • Web Development (Flask)
  • Python Programming
